Al enviar una petición POST a la ruta http://localhost:3000/registro con el siguiente JSON:

{
"nombre": "Maria",
"mensaje": "Hola comunidad"
}

El servidor recibe los datos enviados en req.body y responde con:

{
"estado": "Datos recibidos",
"nombre": "Maria",
"mensaje": "Hola comunidad"
}

Esto confirma que la API recibió correctamente la información enviada por el usuario.
